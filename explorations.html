Video Game Exploration

The game that i made is callled coin collector and the purpose is to have a character collect coins in the time frame that is given, while also trying to have the highest score at the end.

Code for my game 
           WIDTH=400
           HEIGHT=400

           score=0
           game_over=False

           fox=Actor('fox.png')
           fox.pos= 100,100

           coin=Actor('coin.png')
           coin.pos= 200,200


           def draw():
               screen.fill("green")
               fox.draw()
               coin.draw()
               screen.draw.text("score:" + str(score), color="black", topleft=(10,10))
               if game_over:
                    screen.fill("pink")
                    screen.draw.text("Final score: " + str(score), topleft=(10,10), fontsize=60)


           def place_coin():
                coin.x = randint(20,(WIDTH - 20))
                coin.y = randint(20,(HEIGHT - 20))

           def time_up():
                global game_over
                game_over= True

           clock.schedule(time_up, 7.0)

           def update():
                global score
                if keyboard.left:
                fox.x = fox.x - 2

                elif keyboard.right:
                     fox.x = fox.x + 2

                elif keyboard.up:
                     fox.y = fox.y -2

               elif keyboard.down:
                     fox.y = fox.y + 2

               coin_collected = fox.colliderect(coin)

               if coin_collected:
                     sounds.sonic.play()
                     score = score +10
                     place_coin()
